# go-ai-shell

## Requirements
- Ollama (llama3.1) model running locally.
- Go 1.20+


## Installation

```shell
git clone https://github.com/wuilliam321/ai-shell.git
cd go-ai-shell
go build -o ai
```

## Usage

```shell
./ai find all images in png format
```
